  • 21-12-2010 Lezione 3
    Configurazione del web server Apache Httpd. Architettura di Apache.
    • Studiare il manuale di Apache Httpd: Binding, Configuration Files, Configuration Sections (fino a "Virtual Hosts" incluso), Name-Based Virtual Hosts (eccetto la parte su "older browsers"), il modulo prefork.
    • Imparare come fare partire e fermare Apache con "apachectl"
    • Esercizio: installare Apache Httpd su una macchina Linux (per esempio usando il package manager della tua distribuzione preferita. Se non hai una distribuzione preferita suggerisco Ubuntu oppure Debian.)
    • Esercizio: creare un virtual host "pippo.com"
      1. Creare uno o più documenti HTML e salvarli nella directory /tmp/pippo.com
      2. Configurare www.pippo.com per puntare a 127.0.0.1 in /etc/hosts
      3. Creare un file di configurazione con le direttive necessarie per configurare www.pippo.com come un name-based virtual host. Questo file sia salvato come "pippo.conf"
      4. Includere pippo.conf nella configurazione di Apache. Su Ubuntu la maniera migliore è salvare pippo.conf in /etc/apache2/sites-available e poi abilitarlo con "a2ensite pippo"
      5. Verificare che aprendo "www.pippo.conf" nel browser si vedano i file creati al primo passo.
      (Nota che questi esercizi *non* sono facoltativi. Mi aspetto che tu sia in grado di eseguirli. Se ti mancano le informazioni su come fare cerca su Google -- anche questa è una skill essenziale per un professionista. Non è difficile trovare queste informazioni, sono cose molto molto basilari.)

  • 22-12-2010 Lezione 4
    Introduzione alle form HTML. Introduzione alla generazione di pagine dinamiche con CGI.
    • Studiare questo tutorial su HTML Forms.
    • Studiare questa introduzione a CGI.
    • Esercizio: installare, capire e fare funzionare in Apache su Linux gli esempi che ho presentato a lezione:
      • uptime.cgi Mostra informazioni sullo stato della macchina.
      • logo.cgi Dimostra come restituire un'immagine che non è all'interno del DocumentRoot.
      • tempconv.cgi Conversione di temperatura da Fahrenheit a Celsius
    • Esercizio: creare mediante CGI e bash un servizio web che mostri una galleria di immagini.
      1. Salvare alcune foto in una directory. Per ciascuna di esse creare una thumbnail in formato 320x208px e salvarle con nome thumb_NOMEFOTO.jpg
      2. Lo script legge tutti i file nella directory che hanno un nome che corrisponde al pattern thumb*.jpg
      3. Poi lo script restituisce una tabella con 3 colonne che mostra tutte le immagini thumbnail trovate al punto precedente
      4. Se l'utente clicca su una thumbnail deve vedere la versione originale della foto. Questa può essere servita direttamente dal server web, oppure servita da un'altro script cgi nel caso le foto non siano all'interno della DocumentRoot
      Nota bene: questo esercizio richiede una modesta capacità di scripting in bash che dovresti avere acquisito nel corso di Sistemi Operativi. Le competenze in fatto di HTML e CGI dovresti averle acquisite a lezione e sulle letture che ti ho assegnato. Se non sei in grado di risolvere l'esercizio sei già fuori strada per passare l'esame.

  • 11-01-2011 Lezione 5
    Introduzione a Ruby
    • Studiare The Humble Little Ruby Book, capitoli 1, 2, 3, 4 (solo Filesystem interaction)
    • La versione di Ruby che usiamo per queste lezioni è la 1.8.7. Il sistema operativo di riferimento è Linux (ma anche Mac va bene.)
    • Esercizi:
      • Scrivere un programma che conta le occorrenze delle parole nel suo standard input. La versione realizzata in aula ordina alfabeticamente le parole. Esercizio: Modificarla per restituire le parole in ordine decrescente di numero di occorrenze.
      • Codificare la funzione di Fibonacci in maniera efficiente. La versione naive realizzata in aula è troppo lenta per valori > 30. Scrivere un generatore di numeri di Fibonacci, ovvero un oggetto che restituisce ad ogni invocazione di "next" il prossimo numero di Fibonacci: 
                g = FiboGenerator.new
        g.next # => 0
        g.next # => 1
        g.next # => 1
        ...
        Facoltativo (e difficile): calcolare la somma dei numeri di Fibonacci pari e minori di 4 milioni (qui c'è una soluzione).
      • Scrivi un programma Ruby che analizza l'access_log di Apache e restituisce le seguenti informazioni:
        • Elenco delle URL invocate e numero di invocazioni, ordinate per numero di invocazioni
        • Elenco delle URL che hanno provocato errori (status ≥ 400), con il numero di errori, in ordine di numero di errori.
    • Facoltativo: Un'altra guida di riferimento è Programming Ruby, la cui prima edizione è consultabile liberamente online.
    • Facoltativo: imparara Ruby con i Ruby Koans; è un tutorial interattivo. (Per scaricare il progetto da github, installare git (per esempio su Ubuntu: "apt-get install git-core") e poi esegui "git clone git://github.com/edgecase/ruby_koans.git")
    • Facoltativo: una bella collezione di problemi di programmazione, con soluzioni in Ruby: www.rubyquiz.com.
    • Facoltativo: una collezione di utili programmi Ruby di una sola riga.
